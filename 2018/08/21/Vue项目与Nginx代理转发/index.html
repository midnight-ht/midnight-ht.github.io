Vue项目与Nginx反向代理

发表于 2018-08-21 | 分类于 反向代理 ， 跨域 |

  项目开发中用以解决跨域的最佳方案。

Vue

  vue-cli采用http-proxy-middleware插件来进行代理服务器的各项配置。只需在config/index.js文件中利用预留项proxyTable设置地址映射表。

  
    proxyTable: {
        '/api': { // '/api' 转发匹配接口
            target: 'PATH', // 转发地址
            pathRewrite: {
                '^/api': '/'
            }
        }
    }

Nginx

  代理转发

  Nginx反向代理会简单一点，只需在当前服务下加入以下代码块即可。

  
    location /API { // /API 转发匹配的接口
            proxy_pass  PATH; // PATH 服务器地址
    }
